# PostgreSQL with Python using Jupyter Notebook

## Overview

This project demonstrates how to connect Python with a PostgreSQL database using the **psycopg2** library and perform basic database operations in a Jupyter Notebook. It is a simple beginner-level project to understand database connectivity and SQL execution using Python.

---

## Features

* Connect Python to PostgreSQL
* Create a database and table
* Insert records into a table
* Fetch and display records
* Use Jupyter Notebook for interactive development
* Basic GitHub-ready project

---

## Tech Stack

* Python
* PostgreSQL
* psycopg2
* Jupyter Notebook
* Git & GitHub

---

```
postgres-python-jupyter/

```

---

## How to Run the Project

1. Make sure PostgreSQL is installed and running.
2. Open Jupyter Notebook:

   ```
   jupyter notebook
   ```
3. Open `students_db.ipynb`.
4. Run the cells one by one to:

   * Create database
   * Create table
   * Insert data
   * Fetch and display records
